# SBS 게임학원 블로그 Plugin (Jo Jason)

곽두팔의 폭력적인 정보 — 블로그 자동 작성 Plugin.

## 슬래시 명령어

| 명령어 | 용도 |
|---|---|
| `/plan [주제]` | 타겟·키워드·제목 5개·구조 제안 |
| `/write` | 본문 2500자 + 해시태그 20개 + 링크 박스 |
| `/verify` | 체크리스트 실행 |

## 사용 순서

1. `/plan 게임기획 커리큘럼`
2. 제목·구조 선택 ("2번으로")
3. `/write` 자동 실행
4. `/verify` 자동 실행
5. 미통과 수정 → 발행
