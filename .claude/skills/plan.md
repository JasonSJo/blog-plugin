# /plan [주제] — 1단계: 제목·구조 제안

## 실행 내용

주제 또는 키워드를 받으면 아래를 출력하고 대기.

```
📋 1단계 — 제목·구조 제안

▌주 타겟
[B 고등학생 / C 학부모 / A 취업·데뷔 20대] 중 선택 + 이유 한 줄

▌포스팅 유형
[정보성 / 멘토 성장일기 / 메가 / 진성 후기] 중 선택 + 이유 한 줄

▌키워드 전략 (1+2 공식)
- 메인 키워드:
- 연관 키워드 1:
- 연관 키워드 2:
- 검색어 유형: [메인 / 지역+메인 / 일반]

▌추천 제목 5개
1. [정보형]
2. [질문형]
3. [숫자형]
4. [감성형]
5. [직설형]

▌글 구조 outline
- 도입 (약 300자): [곽두팔 공식 — 상황 호명 + 내적 독백]
- 소제목 1: 
- 소제목 2:
- 소제목 3:
- (소제목 4):
- 마무리 (약 400자): [다음 글 예고 + CTA]

→ 어떤 제목·구조로 진행할까요?
```

사용자 선택 대기. 선택받은 후 /write 실행.
