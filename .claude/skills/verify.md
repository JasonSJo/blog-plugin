# /verify — 3단계: 자가 검증

## 체크리스트

**기본 규칙**
- [ ] 본문 글자 수: ___자 (목표 2500±200)
- [ ] 제목 5개 제시
- [ ] 해시태그 정확히 20개
- [ ] 링크 박스 4개 + 색상 다름
- [ ] 첫 문장이 의식의 흐름 아님 (날씨·계절·"안녕하세요" 금지)
- [ ] 한 문단 4문장 이내
- [ ] 한 문장 80자 초과 없음

**SEO**
- [ ] 메인 키워드가 제목에 포함
- [ ] 연관 키워드 2개가 제목에 자연스럽게 녹아듦
- [ ] 메인 키워드가 도입부 첫 3문장에 1회
- [ ] 메인 키워드가 마무리 문단에 1회
- [ ] 메인 키워드 본문 빈도 7~10회
- [ ] 같은 문장에 키워드 중복 없음

**콘텐츠**
- [ ] 주 타겟에 맞는 어휘·예시·CTA
- [ ] 후기성 글은 과거형 시제
- [ ] 곽두팔 도입 공식 적용
- [ ] 과장 표현(100%·최고·1등·혁신적 등) 없음
- [ ] 출처 없는 통계 없음
- [ ] 인용 박스 1개 이상
- [ ] 이모지 없음

## 미통과 처리

```
⚠️ 미통과 [N]개

1. [항목]: [수정 내용]
2. ...
```

미통과 항목만 수정 후 재출력.
모두 통과 시: "✅ 검증 완료 — 발행 전 제이슨 최종 확인 후 게시하세요."
