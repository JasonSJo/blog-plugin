# /write — 2단계: 본문 작성

## 실행 전 확인

/plan 결과에서 제목·구조가 선택되어야 실행. 없으면 /plan 먼저 실행 요청.

## 작성 규칙

### 도입부 (곽두팔 공식 — 가운데 정렬, 줄바꿈)
```
[독자 상황 호명] ~ 하다 보면 / ~ 싶으면
[한 박자 쉬기] 거의 이 생각부터 듭니다.
"[독자 내적 독백]"
```

### 소제목 표기
- 단계 안내: ✓ 접두
- 섹션 헤더: ▌ 접두
- 단계 분해: STEP 01. STEP 02.

### 인용 박스 (각 섹션 끝)
```
"
[핵심 명제 3~4줄, 짧게 끊어서]
"
```

### SEO 키워드 배치
- 도입부 첫 3문장에 메인 키워드 1회
- 본문 자연스럽게 분산 (2500자 기준 7~10회)
- 마무리 문단에 메인 키워드 1회
- 같은 문장에 키워드 중복 금지

### 분량
- 도입 약 300자
- 본문 약 1800자
- 마무리 약 400자
- 총 2500자 ±200자

### 시제
- 후기성·멘토 성장일기: 과거형 (이다 → 였다)
- 정보성: 현재형 OK

### 금지
- "오늘은 ~에 대해 알아보겠습니다"
- "안녕하세요 여러분"
- 혁신적·압도적·유일한·100% 등 과장 형용사
- 출처 없는 통계
- 이모지

### 마무리 구조
1. "이번 글에서는 ___ 정리해봤습니다." 요약
2. "다음 글에서 ___ 다뤄보겠습니다." 예고 (해당 시)
3. 학원 카드 + 약도 이미지 자리 [이미지 삽입 위치 표시]
4. 링크 박스 4개 (색상 다르게)
5. "긴 글 읽어주셔서 감사합니다 :)"
6. 해시태그 20개

### 링크 박스 4개
```html
🟦 게임학원 홈: http://sbsgameclass.co.kr/index.asp
🟩 취업 성공후기: http://sbsgameclass.co.kr/2025/community/interview.asp?txtKind=6
🟧 학원생 포트폴리오: http://sbsgameclass.co.kr/2025/portfolio/portfolio_tch.asp
🟥 상담 신청: https://naver.me/xxoRzqst
```

### 해시태그
게임/웹툰 관련 + 지역명(부산/김해/경남) + 진로 키워드 혼합 정확히 20개.

작성 완료 후 자동으로 /verify 실행.
